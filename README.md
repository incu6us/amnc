### Usage
```bash
NAME:
   create-alert - Create test alert for Alert Manager

USAGE:
   create-alert [global options]

DESCRIPTION:
   This command creates a popup alert for the user.

GLOBAL OPTIONS:
   --alert-duration duration, -d duration            (default: 1m)
   --alert-manager-address string, --address string  The URL of the alert manager service
   --use-tls, --tls                                  Use TLS for the connection to the alert manager (default: false)
   --help, -h                                        show help
```
